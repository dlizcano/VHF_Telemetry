# VHF_Telemetry
Script for VHF load and analyzing radio tracking data. From azimuths to home range plot 

## load packages


```r

library (sp)  # classes for spatial data
library (raster)  # grids, rasters
library (rasterVis)  # raster visualisation
library (maptools)
library (rgeos)
library (sigloc)

```
